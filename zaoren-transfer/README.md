# zaoren-transfer · 规则迁移

将A世界的规则应用到B场景。世界之间的规则跨界应用。

## 安装

```bash
# 已随zaoren系列自动安装
# 或手动安装到全局
git clone <repo-url> ~/.claude/skills/zaoren-transfer
```

## 使用

```
> /ztransfer 芒格格栅思维 → 甲方需求分析

场景：客户说"这个按钮要大一点"

[生成迁移后的分析...]
```

## 迁移示例

```
📍 原始规则（芒格格栅思维）
核心：用多学科的重要模型交叉验证判断

📍 迁移应用（甲方需求分析）

🔬 心理学视角：客户可能不是在说"大小"，而是"找不到"
🏢 商业视角：需要A/B测试数据，否则是拍脑袋
🎨 设计视角：单独放大会破坏系统一致性

💡 建议：追问真实问题 → 提供数据选项 → 展示系统影响
```

## 迁移类型

1. 思想体系 → 具体场景
   - 第一性原理 → 职业规划
   - 逆向思维 → 风险识别

2. 角色视角 → 不同领域
   - 投资人视角 → 择偶决策
   - 产品经理视角 → 装修房子

3. 学科模型 → 日常决策
   - 概率思维 → 要不要跳槽
   - 复利效应 → 学习规划

## 设计来源

从26个爆款SKILL中提取精华：

- zaoren-roundtable 的世界碰撞 → 规则跨界映射
