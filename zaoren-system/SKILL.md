# 造体系

> 一套思想体系，是一个完整的世界——有自己的语言，有自己看不见的东西，有自己的第一性原理。进入它，从内部思考，不是站在外部引用它。

## 核心机制

思想体系不是工具箱——它是一套**认知操作系统**。进入儒家不是引用孔子的话，而是用儒家的语言框架去感知问题。

关键问题：这套体系的第一性原理是什么？它的内在语言是什么？从内部看，什么问题不存在？

## 如何激活一套体系

**第一步：找到那个世界的门**
- 这套体系的核心命题是什么？（能用一句话说清的）
- 这套体系认为什么问题是「错误的问题」？（在那个框架里根本不存在的问题）

**第二步：从内部思考**
- 用这套体系的语言，这个问题是什么？
- 这套体系的第一性原理给出什么方向？

**第三步：标注盲区**
- 从这套体系内部，永远看不见什么？
- 需要哪个补充视角？

## 体系造境的独特价值

**跨体系碰撞是最强的认知工具**：
```
同一个组织问题：
儒家：礼制与修身能解决
奥派：去掉中央控制，让自发秩序运作
系统论：找到反馈回路的结构，改变杠杆点
```

三个体系从不同第一性原理出发，给出根本不同的解决方向。
碰撞点就是那个问题最深的地方。
