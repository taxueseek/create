# zaoren-compare · 世界对比

同一问题在多个世界的处理方式对比。让你看到选择的代价。

## 安装

```bash
# 已随zaoren系列自动安装
# 或手动安装到全局
git clone <repo-url> ~/.claude/skills/zaoren-compare
```

## 使用

```
> /zcompare 老板 投资人 导师

问题：这个创业项目要不要接？

[生成三个世界的对比分析...]
```

## 对比维度

每个世界分析：
1. 核心判断逻辑
2. 关键问题
3. 立场预测
4. 盲区提醒

## 输出示例

```
┌─────────────────────────────────────────────────────────────┐
│ 🎯 老板.world                                                │
├─────────────────────────────────────────────────────────────┤
│ 核心判断：ROI + 资源占用 + 团队影响                           │

│ 他的立场：不接（除非可控）                                    │
│ 关键提醒：关心的是"对团队整体的影响"                          │

┌─────────────────────────────────────────────────────────────┐
│ 💰 投资人.world                                              │
├─────────────────────────────────────────────────────────────┤

│ 他的立场：看情况（非对称收益）                                │

```
