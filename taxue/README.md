# 踏雪问题解决系统（Taxue Problem Solving System）

> 把事情搞清楚，推进行动，固化经验。

## 技能详情

| 技能 | 别名 | 职责 | 触发场景 |
|------|------|------|----------|
| **taxue** | `/taxue` | 入口路由 | 自动推断意图，0秒直达对应工具 |
| **taxue-menu** | `/taxue-menu` | 可视化菜单 | 自动推断失败时的 fallback 菜单 |
| **think** | `/t`, `/think`, `/taxue-think` | 思维澄清 | 方向模糊、纠结、还没想明白 |
| **search** | `/sh`, `/search`, `/taxue-search` | 搜索中枢 | 需要查资料、搜信息、找数据 |
| **solve** | `/ps`, `/solve`, `/taxue-solve` | 问题解决 | 问题清晰，需要具体解法 |
| **breakdown** | `/td`, `/task`, `/breakdown`, `/taxue-breakdown` | 任务拆解 | 复杂任务，需要分步骤、排期、分配 |
| **career** | `/career` | 职场全链路 | 简历、面试、规划、离职 |
| **build** | `/build`, `/taxue-build` | 系统构建 | 写skill、固流程、建系统 |
| **learn** | `/taxue-learn` | 学习方法 | 想学习、整理知识、掌握技能 |
| **taxue-upgrade** | `/taxue-upgrade` | 升级工具 | 升级 taxue skills 到最新版本 |

## 核心原则

1. **先澄清，后解决** —— 问题不清，答案无用
2. **够用就好，不要完美** —— 有限理性，追求当前够用的方案
3. **推动行动，不给建议** —— 不说"你可以考虑"，说"第一步做X"
4. **能固化，不复述** —— 重复的问题写成skill，一次建设长期受益

## 安装

```bash
git clone https://github.com/taxuexunxian/taxue ~/.claude/skills/taxue
```

无需安装任何额外依赖。

**运行环境要求：**

| 依赖 | 说明 |
|-----|------|
| Python 3.6+ | macOS 自带，无需安装 |
| 无浏览器依赖 | 使用对话内菜单 / 终端菜单，零延迟 |

## 文件结构

```

├── SKILL.md           # 系统构建（六步编译法）
```

## 质量自检

每个skill在输出前必须检查：

- [ ] 是否只解决一个主要矛盾？
- [ ] 输出是否直接可用？
- [ ] 用户今天能不能开始做？
- [ ] 是否留了余地让用户参与？
