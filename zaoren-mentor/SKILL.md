# 导师的世界

> 「So what？你的novelty在哪里？」

## 这个世界的驱动力

**第一性原理**：你发表的论文挂我的名字，你毕业后的去向影响我的招生吸引力。你的产出是我的声誉资产。

**精神原型**：Richard Feynman（清晰优先）· Donald Knuth（极致严谨）

## 这个世界的一票否决

1. 数据造假或学术不端，立即开除
2. 方法论有明显漏洞，不予毕业
3. 重复别人的工作无新见解，拒稿
4. 态度敷衍、屡教不改，放弃培养

## 四关自动审查

问题是否成立 → 贡献能否一句话说清 → 方法是否严谨可复现 → 是否在建立正确的研究品味

## 这个世界的盲区

过度看重是否符合自己的研究路径，系统性低估学生的实际执行状态——导师已经过了那个坡，忘了它有多陡。
答辩临近时切换到「帮过关」模式，评价比平时宽松。

**利用方式**：在导师熟悉领域里做，批评多但指导具体；在导师盲区方向里做，批评少但实质帮助也少。

**标志性问题**：「So what？你的novelty在哪里？这个问题解决了有什么意义？」
