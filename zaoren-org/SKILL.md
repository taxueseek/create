# 造组织

> 伟大的组织是思维方式的化石——它在特定约束下演化出来的认知文化，包含了大量无法从外部描述的默会知识。进入它，从内部运作。

## 核心机制

进入一个组织不是「引用它的成功案例」，而是**用那个组织的语言和判断逻辑处理当前问题**。

关键问题：这个组织的第一性原理是什么？它招什么人、拒绝什么人、用什么标准做决策？

## 如何激活一个组织

**第一步：找到那个组织的基因**
- 这个组织的第一性原理是什么？（一句话）
- 这个组织用什么标准决定「要不要做一件事」？
- 这个组织里，什么人被尊重？什么行为被惩罚？

**第二步：从内部运作**
- 把用户的问题放进那个组织，那个组织会怎么拆解它？
- 那个组织的判断标准给出什么答案？

**第三步：标注盲区**
- 这个组织系统性看不见什么？
- 那个盲区在用户的场景里有没有影响？

## 组织造境的独特价值

**不同组织基因处理同一问题，给出根本不同的方案**：
```
如何建研发团队？
贝尔实验室：给最好的人时间和空间，目标是未来20年
麦肯锡：明确问题定义，MECE分解，假设驱动验证
丰田：先建改善文化，研发就是系统性消灭问题的过程
```

选哪个，取决于你的约束条件和核心目标。

配合 zaoren-roundtable 让不同组织基因直接碰撞。
