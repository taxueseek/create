> 一起创造新世界
>
> 最初，我们把这个项目叫造人。
>"你们搞大模型的简直是女娲，捏了同事，捏了前任，捏了老板，捏了导师，最后把自己也捏了一个，替自己上班。"
> 那时候，我们像女娲一样，捏了前任、捏了老板、捏了导师，想让 AI 替我们上班。
但很快我们发现，我们被人这个概念束缚了。于是便把它更名为造物。

单体 agent 的拟人化，只是 AI 使用的初级阶段。
把隐藏在公文、财报、复盘报告背后的决策逻辑和思维方式，跟世间万物的本质提取出来，化作可以自由组合、推演碰撞的引擎，才是我们真正的天命。

## 学说话 vs 学办事

**学说话**——模仿ta的语气、口头禅、说话风格。像个复读机，说出来的话像，但办不了实事。

**学办事**——学会ta怎么判断、怎么决策、怎么评估。像个实习生，能真的帮你预审论文、评估项目、预判甲方反应。

这个工具做的是第二种：不让AI学老板说话，让AI真的能用老板的标准判断——什么方案能过、什么需求要改、什么该直接毙掉。打造一个专属于你的真正的团队

## 造人（三层深度）

素材决定深度。你给什么，就造什么层次的经验：

| 素材质量 | 能造到哪层 | 实际效果 |
|:---------|:---------|:---------|
| 几条聊天记录、描述 | **L1 语气层** | 说话像，但办不了专业的事 |
| 工作文档、代码、邮件 | **L2 能力层** | **能替ta做专业判断（核心）** |
| 书籍、系统方法论 | **L3 智慧层** | 能用ta的思维方式分析新问题 |

**L2 决策树示例**（导师审稿）：

```
张教授的审核流程：
├── 先看 contribution → 不清晰？直接退回
├── 再看 baseline → 不公平？要求补实验
├── 然后看 writing → 语法错误>3个？不看了
└── 最后看 novelty → 只是 incremental？建议转投 workshop
```

你不需要一个AI学张教授说话阴阳你。你需要的是他真的能用张教授的标准帮你审论文——装上这个，Claude 就能。

## 造行业（内行判断）

给行业资料，造出**内行才有的评估能力**。

> 普通分析师：「新能源市场规模500亿，增速15%」
>
> 造出来的行业 skill：「增速是补贴撑的，退坡后约8%。你切的是储能，真壁垒在并网资源。没有电网关系，技术再好也是给别人做嫁衣。三年前12家公司试过，活下来2家。」

---

## 造项目（决策内核）

给项目文档，造出这个项目的**决策逻辑**。

> 普通工具：「任务3逾期了」
>
> 造出来的项目 skill：「这个 feature 的 ROI 算错了。你用的是 MAU，这个场景该看留存。按留存算优先级降两档。参考XX项目教训，这类需求做了留存提升不到 2%。」

---

## 造圆桌（交叉碰撞）

你造了一个投资人、一个行业专家、一个法务。

现在问：「要不要投这家储能公司？」

```
投资人：PE 45倍在这阶段偏高，但如果增速维持...

行业专家：储能真壁垒在并网。看它有没有电网关系。没有就别碰。

项目模型：类比历史，这阶段存活率约30%。关键验证：Q3能不能落地首个并网项目。

--- 碰撞 ---

投资人 → 行业专家：它刚拿了XX电网试点...
行业专家 → 投资人：试点不等于商业化。XX公司2023年也拿过，现在呢？
项目模型：等Q3结果。成功了PE回35以下建仓。失败了永久排除。
```

**人 × 行业 × 项目，交叉碰撞。别的 skill 想都不敢想。**

| skill | 功能 | 场景 |
|:------|:-----|:-----|

## 实际使用示例

### 造一个导师，帮你审论文

```
你：[上传5封导师的审稿邮件]

系统：已提取导师的审稿决策

你：用我导师的视角看看这篇投稿

系统：
- contribution 不够清晰，第三段才点题
- baseline 选了太弱的对比方法，显得提升很大但实际不公平
- 图2 的 legend 太小，看不清
- novelty 属于 incremental improvement，建议 major revision
```

### 造一个行业，评估投资机会

```
你：[上传10份储能行业研报、3个案例]

系统：已构建储能行业评估框架

你：评估这家储能公司

系统：
- 技术没问题，但并网资源是短板
- 创始人有电网背景，这是核心优势
- 但商业化进度落后竞争对手6个月
- 建议：等Q3首批并网项目落地后再评估
```

### 启动圆桌，多方辩论

```
你：让投资人、行业专家、法务讨论这个并购案

投资人：估值合理，但整合风险高...
行业专家：标的技术在行业内排前三，但客户集中度太高...
法务：知识产权有瑕疵，需要尽调...

--- 交叉质询 ---

投资人问法务：知识产权瑕疵对估值影响多大？
法务：如果核心专利有争议，估值应该打7折...
行业专家：其实技术本身不难复制，关键在人...
```

## 使用之前，你要知道

这个工具像模具——帮你把经验固化成可复用的形状，但造出来的东西好不好，还得看原材料。

**素材决定质量。** 你给的是聊天记录，造出来的就是说话像；给的是工作文档和决策邮件，造出来的才能办专业的事。Garbage in, garbage out。

**不能替代真人。** 造出来的导师能帮你预审论文，但最终签字负责的还是真人导师。造出来的投资人能帮你分析项目，但投不投、投多少，决定权在你。

**有边界。** 适合用来：分析判断提效、预审预演、多方视角碰撞。不适合用来：骚扰、冒充、欺骗、替代真人做最终决策。

**一句话：工具帮你思考和准备，但责任在你，真人不可替代。**
