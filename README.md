## 介绍

Create 是一套**认知世界的工具**——把隐藏在经验、文本、决策背后的逻辑，转化为可复用的 AI 工具。

不是让 AI 学说话，是让 AI 学办事。

## v4.0 核心升级

### 1. 三层架构（新增）

v4.0 最大的变化是从单一「造人」扩展到完整认知链路：

```
┌─────────────────────────────────────────┐
│  ming v2.0  洞察层 —— 看透「是什么」      │
│  中华思考者 · 哲思伙伴 · 思维兵器         │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  zaoren v4.0 决策层 —— 判断「怎么做」     │
│  7个决策世界 + 3个组织工具 + 3个支撑工具   │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  taxue v3.0 执行层 —— 解决「动手做」      │
│  快速解法 · 任务拆解 · 系统构建 · 版本管理 │
└─────────────────────────────────────────┘
```

**使用流程**：
```
遇到问题 → ming 想透 → zaoren 决策 → taxue 执行
```

### 2. 统一世界格式（重写）

所有 zaoren 角色层 skill 现在遵循统一结构：

| 模块 | 内容 | 目的 |
|:---|:---|:---|
| 公理 | 5条第一性原理 | 定义这个世界的底层逻辑 |
| 语言 | 关键词汇 | 快速识别进入哪个世界 |
| 硬规则 | 规则/含义/判断标准 | 可执行的判断依据 |
| 盲区 | 结构性盲区+利用方式 | 暴露局限，避免盲从 |
| 输出格式 | 标准化模板 | 保证输出质量一致 |

### 3. 技能精简（-51%）

从 35 个 skill 精简到 17 个，降低认知负荷：

| 处理方式 | 旧技能 | 新归属 |
|:---|:---|:---|
| **合并** | zaoren-compile | → taxue-build 模式B |
| **并入** | naval-mindset | → zaoren-naval |
| **降级** | zaoren-troll/reviewer | → roundtable 压力测试模式 |
| **删除** | zaoren-era/org/system | 低频，ming 可覆盖 |
| **保留** | 核心 7 世界 + 3 组织 + 3 工具 | 独立 skill |

### 4. 双模式编译（增强）

taxue-build v2.0 现在支持两种构建模式：

```bash
/build --solution    # 从解法编译 → 个人模板/SOP（15分钟）
/build --persona     # 从人物编译 → 认知世界/完整Skill
```

### 5. 版本管理（新增）

taxue-upgrade —— 一键管理所有 skill 版本：

```bash
/upgrade              # 检测可更新
/upgrade all          # 升级全部
/upgrade --rollback   # 回滚到备份
```

## 七世界速查

| 世界 | 核心问题 | 适用场景 |
|:---|:---|:---|
| **boss** | "谁负责？最坏情况？" | 向上管理、资源申请 |
| **investor** | "市场规模？为什么是你？" | 项目评估、创业决策 |
| **munger** | "反过来想会怎么失败？" | 复杂决策、风险识别 |
| **drucker** | "这件事该不该做？" | 战略取舍、放弃决策 |
| **client** | "出了问题谁背锅？" | 需求分析、交付沟通 |
| **mentor** | "3年后会后悔吗？" | 职业选择、长期规划 |
| **naval** | "杠杆在哪里？能复利吗？" | 财富积累、个人成长 |

## 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/taxueseek/create.git

# 安装 zaoren 系列
cp -r create/zaoren* ~/.claude/skills/

# 安装 taxue 系列
cp -r create/taxue* ~/.claude/skills/
```

### 使用

```bash
# 进入某个决策世界
/zaoren-boss 帮我看这个方案的风险

# 多视角对比
/zaoren-compare 老板 投资人 导师

# 构建自己的工具
/taxue-build --solution   # 从解法编译
/taxue-build --persona    # 从人物编译

# 升级技能
/taxue-upgrade
```
